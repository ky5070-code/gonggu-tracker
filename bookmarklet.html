북마크릿 설치

위의 핑크 버튼을 브라우저 북마크바로 드래그해서 저장하세요. 한 번만 하면 됩니다!

2

공구 찾기

인스타그램, 네이버 블로그 등에서 공구 포스팅을 발견하면 해당 페이지를 열어두세요.

3

북마크릿 클릭

북마크바의 "🔖 공구 추가 북마크릿" 버튼을 클릭하면 입력 폼이 뜹니다. 현재 페이지 URL이 자동으로 채워집니다.

4

정보 입력 후 저장

인플루언서, 상품명, 가격, 마감일 입력 후 "저장하기" 클릭. 처음 한 번만 GitHub 토큰 입력이 필요합니다.

5

사이트에 자동 반영

매일 오전 8시에 자동 업데이트되거나, 저장 후 뜨는 링크를 클릭하면 즉시 업데이트 가능합니다. 직접 추가한 공구는 ✅ 실제 공구 배지로 표시됩니다!

📌 GitHub 토큰 발급 방법
처음 저장할 때 토큰이 필요합니다. 한 번 입력하면 브라우저에 저장돼서 다음엔 안 물어봐요.

1. github.com/settings/tokens/new 접속
2. Note: 아무 이름 (예: gonggu)
3. Expiration: No expiration
4. repo 체크박스 선택
5. Generate token 클릭 → 복사
